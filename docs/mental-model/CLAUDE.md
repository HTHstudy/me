# Mental Model 문서 작성 규칙

## 목적

- 정답을 제시하지 않고 문제를 해석하는 틀을 제공한다.

## 형식

- 문서 상단에 제목과,
  이 문서가 독자의 사고를 어디에 고정시키는지
  드러내는 질문형 인용구를 둔다.

## 금지

- 결론, 처방, "정답"을 문서 초반에 제시하지 않는다.
- 독자가 무엇을 생각해야 하는지 단정하지 않는다.

## 작성 관점

- 특정 선택이 만들어내는 상황, 작용하는 힘, 생기는 긴장과 트레이드오프를 묘사한다.
- "왜 이런 선택을 하게 되는가"를 중심 질문으로 둔다.
- 규칙이나 결론처럼 들리는 문장은,
  관찰 가능한 현상이나 반복되는 패턴의 형태로 다시 쓴다.

## 위반 신호

- 문장이 맥락 없이 "정답 문장"으로 인용 가능하면 위반이다.
- "~해야 한다 / ~가 옳다"로 요약 가능한 문장이 반복되면 위반이다.
