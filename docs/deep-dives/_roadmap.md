## Decisions 문서와의 연결

Deep Dives 문서는 Decisions 문서에서 참조될 수 있습니다.

| Deep Dives | 연결되는 Decisions |
|------------|-------------------|
| 제어 vs 비제어 | 이 컴포넌트는 제어되어야 하는가 |
| 합성 패턴 | 이 컴포넌트는 나눠야 하는가 |
| React Context | 상태를 어디에 둘 것인가 |
| 리렌더링과 최적화 | (성능 이슈 발생 시 참조) |

## 메모

- Deep Dives는 "결정"이 아니라 "이해"에 초점을 맞춘다
- "이렇게 쓰면 된다"가 아니라 "이것이 왜 이런 형태인가"를 탐구한다
- Decisions 문서에서 개념 이해가 필요할 때 참조된다
